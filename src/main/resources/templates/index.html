Connect Sheet API

Interaja com o Google Sheets usando Java e Spring Boot

Funcionalidades da API

A API permite realizar as seguintes operações:

Response:
[
    {
        "id": "523065a4-8c01-4d50-9603-3e57cf18e77f",
        "name": "John Doe",
        "email": "john@doe.com"
    }
]

Response:
{
    "id": "523065a4-8c01-4d50-9603-3e57cf18e77f",
    "name": "John Doe",
    "email": "john@doe.com"
}
                    

Body:
{
    "name": "John Doe",
    "email": "john@doe.com"
}
Response:
{
    "id": "523065a4-8c01-4d50-9603-3e57cf18e77f",
    "name": "John Doe",
    "email": "john@doe.com"
}

Body:
{
    "name": "John Doe",
    "email": "john@doe.com"
}
Response:
{
    "id": "523065a4-8c01-4d50-9603-3e57cf18e77f",
    "name": "Updated Name",
    "email": "updated@email.com"
}

Response:
{
    "message": "Usuário removido com sucesso"
}

How To: Configuração e Testes

Veja como configurar o projeto na sua máquina:

  1. Certifique-se de ter o Git instalado.
  2. Execute o comando: git clone https://github.com/lucasmoraist/connect-sheet.git.
  3. Entre no diretório do projeto: cd connect-sheet.

  1. Acesse o Google Cloud Console.
  2. Crie um projeto e habilite a API Google Sheets.
  3. Gere credenciais do tipo OAuth 2.0 Client ID.
  4. Baixe o arquivo credentials.json e salve em src/main/resources.

  1. Certifique-se de ter o Java 11+ e o Maven instalados.
  2. Instale as dependências executando: mvn clean install.
  3. Execute o projeto com: mvn spring-boot:run.

  1. Utilize ferramentas como Insomnia ou curl para testar os endpoints.
  2. Configure as requisições conforme as funcionalidades: GET, POST, PUT, DELETE.
